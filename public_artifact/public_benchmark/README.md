# MetricCaliberBench Public Benchmark Directory

This directory contains the public benchmark layers used by the CaliberGraph submission:

- `bird_metric_caliber/`: BIRD-derived aggregate diagnostics.
- `iowa_liquor_metric_caliber/`: real public row-level business data benchmark.
- `data/`: Chinook-MetricCaliber public SQLite stress benchmark files.
- `govtwin_metric_caliber/`: public anonymized semantic-twin stress benchmark.
- `multigov_metric_caliber/`: public anonymized production-governance benchmark.
- `industrial_case_text_metric_caliber/`: real desensitized enterprise case text benchmark.

Prediction protocols use blind files and public catalogs/policies. Scorer-only gold labels are separated where blind/gold splits are released.
