# MultiGov-MetricCaliber

MultiGov-MetricCaliber is an anonymized public industrial-governance benchmark derived from production DataGov artifacts.
It releases typed semantic catalogs, caliber/policy/temporal labels, provenance categories, benchmark queries, and deterministic evaluation outputs.
It does not release raw enterprise rows, raw source names, private table names, private column names, or private-to-public mappings.

## Files

- `domain_catalog.jsonl`: anonymized domains and aggregate counts.
- `metric_catalog.jsonl`: anonymized governed metric/policy signals.
- `dimension_catalog.jsonl`: anonymized dimensions and hierarchy.
- `governance_edges.jsonl`: typed anonymous edges.
- `policy_catalog.jsonl`: public refusal and temporal verification policies.
- `source_provenance.jsonl`: provenance categories and regenerated public provenance IDs.
- `physical_coverage.jsonl`: anonymous physical-asset coverage records.
- `metric_coverage_bindings.jsonl`: metric-specific dependency-to-asset bindings.
- `test_cases.jsonl`: public NL2Metric-Caliber cases and labels.
- `release_audit.json`: conflict and leakage scan summary.
- `results/`: deterministic baseline predictions and results.

## Release Boundary

- Public: anonymous IDs, broad domain type, metric role, dimension hierarchy, policy category, provenance category, labels, and evaluator outputs.
- Withheld: raw enterprise rows, raw private queries, source table names, source column names, product/customer/employee identifiers, and private-to-public mapping.
